Whirlwind Mixer Frontend (HW6 bonus)

Minimal UI to interact with your deployed Whirlwind contract. It listens to the Deposit event to stay in sync with the on-chain Merkle root. For proofs, paste the hex proof bytes (0x...) and the expected public inputs are assembled by the contract.

Tip: select HW6/CS595Lab/hw6/contracts/deposit_circuit/target/proof. The UI will auto-extract the proof bytes in the format the on-chain verifier expects.
Your contract constructs public inputs as: [currentRoot, newRoot, commitment, depositIndex].
